# intergado

Este projeto é resultado do processo seletivo para o cargo de Engenheiro de Software na Ingergado.

O projeto tem os seguintes requisitos:

```
Crie um projeto com a versão mais estável do Spring Boot usando Maven; 

Os requisitos a seguir envolvem banco de dados. Utilize o H2. 

Crie uma classe/entidade para representar uma fazenda, com um id incrementado automaticamente e o nome da mesma; 

Crie uma classe/entidade para representar os animais da fazenda, contendo um id incrementado automaticamente, a fazenda e um campo string representando a tag (identificação) do animal; 

Crie endpoints para as operações de CRUD nessas entidades, consumindo e retornando no formato JSON; 

Crie um endpoint para receber uma lista de animais e persistir todos de uma vez. Caso haja algum erro nesse processo, nenhum animal deve ser persistido; 

Incorpore o Swagger ao projeto, incluindo o SwaggerUI. O teste das funcionalidades solicitadas será realizado por ele; 

Registre logs na saída padrão (stdout) contendo informações relevantes para debug; 

Crie um repositório público no Github com o código solicitado e envie o link para o endereço “luigi.cavalcanti@intergado.com.br” dentro do prazo acordado. Seu código será avaliado no prazo máximo de três dias. Após esse período você pode excluir o repositório; 

Certifique-se que o código executa sem erros no branch master. 
```
